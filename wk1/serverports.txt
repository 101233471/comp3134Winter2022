What is the definition of server port?

	A Server Port is a number thats used to identify a transaction over
a network by specifying both the host and the service.

• What is server port range?

	Server port numbers range from 0 to 65535 and port numbers 0 to 1023 are reserved 
for privilaged services and designated as well-known ports. The port numbers specify the port 
used by the server.
